# Knee Rehab Telegram Mini App

Мини‑приложение Telegram для реабилитации колена: пошаговая тренировка, таймер/повторы, календарь, геймификация и конструктор упражнений.

## Реализовано

- Пошаговый режим выполнения с кнопками `Назад / Готово / Пропустить`.
- Таймер для удержаний и счетчик для повторов.
- История сессий + календарь отмеченных дней.
- XP, уровень и бейджи (геймификация).
- Конструктор упражнений:
  - добавление/удаление упражнений;
  - поддержка YouTube-ссылки;
  - если YouTube не указан, показывается встроенная CSS-анимация.
- Telegram WebApp hooks: `ready`, `expand`, haptic и alert.

## Быстрый запуск

```bash
npm start
```

Открой `http://localhost:3000`.

## Как редактировать упражнения

1. Нажми **«Редактировать упражнения»**.
2. Заполни форму:
   - `Название`;
   - `mode`: `time` или `reps`;
   - `Подходы`;
   - `Секунды` (для `time`) или `Повторы` (для `reps`);
   - `YouTube URL` (необязательно);
   - `Описание движения`.
3. Нажми **«Добавить упражнение»**.

Если в упражнении нет YouTube, приложение автоматически показывает сгенерированную анимацию движения.

## Подключение к Telegram

1. Создай бота через [@BotFather](https://t.me/BotFather).
2. Разверни приложение на HTTPS-домене.
3. В BotFather выполни `/setmenubutton` и укажи URL приложения.

## Важно

Это трекер, а не медицинская диагностика. При усилении боли/отеке/блокировке сустава прекрати упражнения и обратись к врачу.

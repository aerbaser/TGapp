# Knee Rehab Telegram Mini App

Готовое мини-приложение для Telegram Web Apps: пошаговое прохождение упражнений для колена с таймером, подходами, повторами, историей и простым трекингом прогресса.

## Что внутри

- 10 упражнений из твоего списка ссылок.
- Режим **time** (таймер 45с) и **reps** (счетчик повторов).
- Пошаговый flow: `Назад / Готово / Пропустить`.
- История выполненных сессий (LocalStorage).
- Подсказка следующей тренировки через день.
- Haptic feedback и alert внутри Telegram WebApp (если открыт в Telegram).

## Быстрый запуск

```bash
npm install
npm start
```

После запуска открой `http://localhost:3000`.

## Подключение к Telegram боту

1. Создай бота через [@BotFather](https://t.me/BotFather).
2. Размести приложение на HTTPS-хостинге (например, Railway, Render, Fly.io, Vercel).
3. В BotFather:
   - `/setmenubutton`
   - выбери бота
   - URL поставь на твой HTTPS адрес (например `https://your-domain.app`)
4. Открой бота и жми кнопку меню — загрузится мини-апп.

## Где настраивать упражнения

Список находится в `app.js` в массиве `exercises`.

## Важно

Приложение не заменяет врача. Если усиливается боль, появляется отек, "заклинивание" сустава или нестабильность — останови упражнения и обратись к специалисту.
